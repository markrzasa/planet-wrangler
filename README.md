# planet-wrangler

A [piston](https://github.com/PistonDevelopers/piston)-based game.

***This game requires a PS5 clicker to play.***

## Build and Run

Build the game:
```shell
cargo build
```

Run the game:
```shell
cargo run
```

## Lint

Install [Clippy](https://github.com/rust-lang/rust-clippy). This needs to be done once.
```shell
make install-clippy
```

Lint with Clippy:
```shell
make lint
```
